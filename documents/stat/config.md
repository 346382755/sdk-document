配置接口说明
=========================

配置接口日志:
```
   获取各种游戏配置表数据
```

POST `http://beacon.ztgame.com/game/config`
 
### 接口参数
 
| 参数名 | 说明 | 范例 |
|------|------|------|
| items | 道具配置 | 1,23,232,32 |
| stages | 关卡配置 | 1,1,1,1,11 |
| events | 事件配置 | 1,1,1,2,2 |
| tasks | 任务配置 | 12,1,232,3 |


### 对接参数范例

```
items=1,2,2,2&stages=1,2,23,2&events=1,1,1,1,1&tasks=12,2,3,1
```
